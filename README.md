

# ✨ YatraAI – AI Travel Planner (Frontend)

YatraAI is a modern, visually engaging frontend web application designed to make travel planning intuitive and inspiring.

Built for **Frontend Frenzy**, this project focuses on **creative UI/UX design**, smooth interactions, and a seamless user experience.

## 🚀 Features

## 🎯 Design Highlights

- **Typography Focus** → Large, bold hero text for impact  
- **Color Strategy** → Dark theme with gradient highlights for premium feel  
- **User Flow** → Simple and guided interaction (search → explore → plan)  
- **Minimalism** → Reduced clutter for better usability
